Neste Projeto 01, criamos uma calculadora que realiza as operações básicas.

Criamos um arquivo html e organizamos ele para receber e demonstrar o que o JavaScrit fará.

Também criamos um arquivo CSS, para estilizar a página e deixá-la agradável a nossos olhos.

Após isso, criamos a index.js , onde criamos um função ela recebe os valores digitados pelos usuários, além da "box" que foi escolhida para decidir o cálculo a ser feito.
Enfim, retornamos o valor do cálculo com um innerHTML.
